This project contains the gopdfrab PDF/A verifier and converter. PDF/A-1b conversion is fully implemented and has no opt-ins: Convert always produces a conformant rewrite for any resolvable graph, rasterizing a page as an automatic last resort when no in-place fixer can repair it.

sentences, KISS principle, maximum of 300 characters. Always add comments to exported names and functions, in other 
cases only when the comment describes a tricky or  complicated case, never describe technical details in comments, 
focus on the "what" not the "how".
