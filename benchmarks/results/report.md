# gopdfrab benchmark report
Generated by `benchmarks/scripts/report.py`. See `benchmarks/README.md` for methodology, caveats, and what each metric measures. Two regimes are reported separately and should always be read together: **cold** (includes process/VM startup — realistic for one-shot CLI use) and **batch** (amortized across the whole corpus in one process — isolates the verification engine).
